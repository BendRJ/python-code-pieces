# Python Code Pieces

A collection of Python examples demonstrating core language features and practical applications.

## Contents

### `generator_example.py`
Demonstrates how generators use `yield` to pause and resume function execution.

### `decorator_example.py`
Shows how to use decorators to add behavior to functions without modifying their code.

### `contextmanager_decorator.py`
Explains the `@contextmanager` decorator for creating custom context managers with setup/cleanup logic.

### `monte_carlo_sim.py`
Performs a Monte Carlo simulation on occurrences of a specific card type in an opening hand of 7 in a given deck list.
